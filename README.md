🌸 Iris Flower Classification
This project is a Machine Learning classification model built using the famous Iris dataset. It predicts the species of iris flowers based on features like sepal length, sepal width, petal length, and petal width.

📌 Project Overview
The goal of this project is to classify iris flowers into three species:
1.Setosa
2.Versicolor
3.Virginica
This project demonstrates a complete Machine Learning workflow, from data preprocessing to model evaluation.

🚀 Algorithms Used
1.Support Vector Machine (SVM)
2.Logistic Regression
3.Decision Tree

📊 Accuracy
The model achieved approximately 93% accuracy on the test dataset.

🛠️ Technologies & Libraries
1.Python
2.Pandas
3.NumPy
4.Scikit-learn
5.Matplotlib

🔄 Workflow
This project includes the following steps:
1.Data Loading
2.Data Preprocessing
3.Train-Test Split
4.Model Training
5.Model Evaluation
6.Predictions on New Data

📷 Visualization
Matplotlib is used to visualize the dataset and understand feature relationships.
